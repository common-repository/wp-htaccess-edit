Simple and Safe WordPress htaccess file editor.

== Description ==

WP Htaccess File Edit provides a safe, simple and fast way to edit the WordPress site’s .htaccess file from admin panel.

Access WP Htaccess Edit via the “Htaccess” menu in WP admin.

== Installation ==

1. Extract the zip into the *wp-content/plugins* directory
2. Activate the plugin through the Admin panel of your WordPress
3. Go to Menu -> Htaccess
